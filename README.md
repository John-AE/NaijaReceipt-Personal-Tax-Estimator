# NaijaReceipt Personal Tax Estimator 2025

A modern, high-performance personal income tax estimator for Nigeria, fully compliant with the **Nigeria Tax Act (NTA) 2025**.

## Features
- **Accurate Calculations**: Computes tax based on the latest graduated tax bands (0% to 25%).
- **Benefits-in-Kind (BIK)**: Handles housing and asset perks provided by employers.

- **Regulatory Guidelines**: Integrated knowledge base for understanding tax jargon and rules.
- **Responsive Layout**: Premium, light-themed design with a sticky header and sidebar navigator.

## Tech Stack
- React 19
- TypeScript
- Vite
- Framer Motion (Animations)
- Lucide React (Icons)
- Vanilla CSS (Glassmorphism & Professional UI)

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
